Overview

Fine-tuned YOLOv8 deep learning model on custom thermal imaging dataset achieving 89.6% mAP50 and 86.7% precision for human detection in low-visibility fire evacuation scenarios. The system is designed to identify trapped individuals in smoke-filled environments where traditional camera systems fail.

Developed a Flutter mobile application with integrated video processing pipeline and YOLOv8 inference engine for post-incident rescue analysis. The application enables first responders to quickly analyze thermal footage and locate individuals in emergency situations.

Technical Implementation

  • Fine-tuned YOLOv8 model on custom thermal imaging dataset with data augmentation
  • Achieved 6-8ms inference speed per frame for near real-time detection capabilities
  • Developed Flutter mobile application for cross-platform deployment (iOS/Android)
  • Implemented video processing pipeline for frame extraction and analysis
  • Integrated TensorFlow Lite for on-device model inference
  • Optimized model for mobile deployment with quantization techniques
  • Created custom dataset annotation pipeline for thermal imaging data

Technologies Used

YOLOv8 Flutter Python TensorFlow Lite Computer Vision Thermal Imaging OpenCV Dart

Challenges & Solutions

  • Low Visibility Detection: Trained model specifically on thermal imaging data to detect heat signatures in smoke-filled environments
  • Mobile Performance: Optimized model using TensorFlow Lite quantization to achieve 6-8ms inference on mobile devices
  • Dataset Limitations: Created custom thermal imaging dataset with diverse fire scenarios and human positions
  • Real-time Processing: Implemented efficient video processing pipeline to handle frame extraction and analysis
  • Cross-platform Deployment: Used Flutter framework to ensure consistent experience across iOS and Android

Results & Impact

The system achieved 89.6% mAP50 and 86.7% precision on thermal imaging test dataset, demonstrating high accuracy in detecting humans in low-visibility conditions. The 6-8ms inference speed enables near real-time processing on mobile devices, making it practical for emergency response scenarios.

The Flutter mobile application provides an intuitive interface for first responders to analyze thermal footage from fire scenes, potentially saving lives by quickly identifying trapped individuals in post-incident rescue operations. The system represents a significant advancement in emergency response technology for fire evacuation scenarios.
